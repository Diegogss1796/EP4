# PA4 - Parte 3: Modulo publico en Next.js

Proyecto Next.js independiente para el modulo publico del catalogo de cursos.

## Rutas

- `/` - Home publico
- `/catalogo` - Catalogo publico de cursos
- `/catalogo/[id]` - Detalle dinamico de curso con `generateStaticParams`

## Ejecucion

```bash
npm install
npm run dev
```

El modulo usa el puerto `3001` para no interferir con la app principal en Vite.

## Scripts

```bash
npm run dev
npm run build
npm run start
```
